# Mortgate Calculator API

## Pre-requisites

* Node v12+
* Npm v6+
* Postman

## Get Started

* To run locally

`npm install`
`npm run local`

## Deploy to Heroku

* Make sure you push the latest changes

`git push heroku main`

* Open the app in your default web browser

`heroku open`

## Testing

`npm run test:unit`

`npm run test:integration`
